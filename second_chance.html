Second Chance

Second Chance is a final year project that myself and 10 classmates are working on. We're creating a 'hack and slash' game with 'time-travel' elements in Unreal Engine 5.3.2. This project has been a great opportunity for me to learn workplace standards for developing with such a large team. We use agile development techniques such as a ticket system via GitHub Issues and feature branches. We are currently using 'sprints' in our development.
